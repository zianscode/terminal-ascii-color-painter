# **ASCII Colored Image Renderer**

Project Python ini digunakan untuk mengubah sebuah gambar menjadi tampilan ASCII berwarna di dalam terminal.
Setiap pixel gambar akan digantikan oleh karakter acak (huruf/angka/simbol) namun tetap mempertahankan warna asli dari gambar.

* Mendukung hampir semua format gambar (JPG, PNG, JPEG, dsb.)
